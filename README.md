# Linux Security Labs

This repository contains hands-on labs and notes created during my study of Linux and basic Cyber Security concepts.

## Topics covered
- Linux filesystem navigation
- File permissions and ownership (chmod, chown, ls -l)
- Users, groups, sudo and root concepts
- Risks related to wrong permissions and executable files
- Introduction to SUID binaries (educational purpose)
- Virtual machines with VirtualBox
- Kali Linux basic usage

## Environment
- Kali Linux (VirtualBox)
- Ubuntu
- Bash shell

## Purpose
This repository is intended to demonstrate my practical learning path as a Junior IT / IT Support with interest in Cyber Security.
